# Magic Methods Documentation

#### Magic Methods
- Spesial method bawaan dari PHP yang membantu untuk mengelola object.

##### List Magic Methods:
- __construct() 
- __destruct()
- __call()
- __callStatic()
- __get()
- __set()
- __isset()
- __unset()
- __sleep()
- __wakeup()
- __serialize()
- __unserialize()
- __toString()
- __invoke()
- __set_state()
- __clone()
- __debugInfo()
